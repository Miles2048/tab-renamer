# Tab Renamer Pro 🚀

一款极致极简、智能化的标签页重命名浏览器扩展。通过 AI 建议和持久化标题，深度定制您的工作空间。

![预览](screenshot.png)

## ✨ 功能亮点

- **🤖 多模型 AI 支持**：内置 Google Gemini, OpenAI GPT-4o, Anthropic Claude, 以及 OpenRouter (支持 DeepSeek/Llama)。
- **🌗 双色模式**：极致纯白的浅色主题与深邃的暗黑模式，支持一键切换并自动保存偏好。
- **📁 Emoji 库**：精选常用 Emoji，让标签页分类更有序且美观。
- **💾 强持久化**：自定义标题在页面刷新、浏览器重启后依然有效。
- **⚡ 顶级交互体验**：
  - **App 级丝滑转场**：设置面板采用平滑滑块切换效果。
  - **高效命名**：回车确认后自动关闭界面（仅限手动重命名）。
  - **一键恢复**：分秒间找回原始网页标题。
- **⌨️ 快捷键支持**：使用 `Cmd/Ctrl + Shift + Y` 极速唤起。

## 🚀 安装指南

1.  克隆本仓库或下载压缩包。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  开启右上角的 **开发者模式**。
4.  点击 **加载已解压的扩展程序**，选择本目录。

## 🛠️ 配置说明

1.  点击设置图标（齿轮）。
2.  进入 **AI Configuration** (AI 设置)。
3.  选择您喜爱的 AI 供应商并粘贴 API Key。
4.  您的 Key 安全存储在 Chrome 账号同步空间中，加密且私密。

## 🛡️ 隐私与安全

- **端到端通讯**：AI 请求直接从您的浏览器发往各官方接口，不经过任何中转服务器。
- **本地化存储**：API Key 和标题数据仅保存在您的本地/Google 账号中，开发者无法获取。
